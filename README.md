# Prácticas de SQL - Base de Datos "Jardinería"

Este repositorio contiene la solución a una guía de **88 ejercicios de consultas SQL** resueltos sobre una base de datos de una empresa de jardinería.

## Contenido del Proyecto

El proyecto incluye tres scripts que deben ejecutarse en este orden:

1. **`estructura.sql`**: Creación de las tablas y relaciones (claves primarias y foráneas).
2. **`altas.sql`**: Inserción de los datos de prueba.
3. **`consultas.sql`**: Enunciados y resolución de las 88 consultas (filtros, joins, agrupaciones y subconsultas).

## Cómo Usarlo

Para utilizar la base de datos localmente (en MySQL / MariaDB), ejecutá los scripts en el siguiente orden:

```sql
SOURCE estructura.sql;
SOURCE altas.sql;
SOURCE consultas.sql;
